# 🗳️ Soroban Voting DApp
## 📌 Project Description

This project is a simple decentralized voting application built using Soroban smart contracts on the Stellar network. It allows users to vote on predefined proposals in a transparent and tamper-proof way.

The goal of this project is to demonstrate how on-chain governance or polling systems can be implemented using Soroban.

---

## ⚙️ What it does

- Initializes a list of proposals
- Allows users to vote for a proposal
- Stores vote counts on-chain
- Retrieves individual or all proposal data

---

## 🚀 Features

- 🧾 **On-chain proposal storage**  
  All proposals and vote counts are stored securely on the blockchain.

- 🗳️ **Simple voting mechanism**  
  Users can vote for any proposal by its ID.

- 📊 **Real-time vote tracking**  
  Fetch vote counts anytime directly from the contract.

- 🔍 **Queryable data**  
  Retrieve a single proposal or all proposals.

- ⚡ **Lightweight and efficient**  
  Designed to be minimal and gas-efficient using Soroban SDK.

---
